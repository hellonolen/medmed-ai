# medmed.ai — Health Research Platform (2026)

**medmed.ai** is a conversational health research platform designed to provide accessible, research-assisted information.

## Core Features
- **Research Dashboard**: A central location for health inquiries.
- **Symptom Research**: Analysis of reported issues for informational purposes.
- **Safety Checks**: Interaction reviews for medications.
- **Visualization**: Visual analysis for research-assisted reports.

## Technology
- **Frontend**: Vite + React + TypeScript (located in `/v2`)
- **Backend**: Cloudflare Workers (located in `/workers/medmed-agent`)
- **Intelligence**: Multi-source research conductor powered by Gemini and other leading models.

## Deployment
- **Frontend**: Deployed to Cloudflare Pages.
- **Backend**: Deployed to Cloudflare Workers (api.medmed.ai).
